Fableau

A Tableau Web Data Connector for Facebook page data.

Please select the Facebook pages you would like to retrieve data for and click the big purple button.

For more information on how Fableau.io works and how you can use it please read our documentation. Fableau.io built by @RobDWaller, please feel free to connect and send any questions you have. @copy; Rob Waller 2017, released under the MIT License.
